🎧 Spotify Integration

Due to Spotify's API terms, direct audio streaming requires a Spotify Premium account and OAuth authentication. To enable full integration:

1. Create an app at developer.spotify.com
2. Add your Client ID to this app's config
3. Set redirect URI to this page's URL
4. Users log in via Spotify OAuth — the Web Playback SDK then streams audio directly into the visualizer

In the meantime, paste a Spotify track URL below and we'll open it in Spotify while the visualizer runs in demo mode.
